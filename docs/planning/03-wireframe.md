# Memoria — 와이어프레임 설계서

> 작성일: 2026-04-30
> 기반 문서: 01-feature-definition.md (P0 + P1 기능 범위)
> 컬러 팔레트: Deep Navy #1B2A4A (Primary) / Warm Gold #C8A96E (Accent) / Off White #FAF6F1 (Background)
> 기준 해상도: 데스크톱 1440px / 모바일 breakpoint 768px

### 구성 요소

| 요소 | 설명 |
|------|------|
| 로고 | Memoria 브랜드 로고 + 서브 카피 |
| 이메일 입력 | placeholder: "이메일 주소", 유효성 검증 (이메일 형식) |
| 비밀번호 입력 | placeholder: "비밀번호", 보기/숨기기 토글 아이콘 |
| 로그인 유지 | 체크박스 (Refresh Token 30일 활용) |
| 로그인 버튼 | Deep Navy 배경 + Off White 텍스트, 전체 너비 |
| 소셜 로그인 | 카카오(노란색) / 구글(흰색+테두리) / Apple(검정색) 각 브랜드 가이드 준수 |
| 비밀번호 찾기 | P1 기능, 텍스트 링크 |
| 회원가입 링크 | 회원가입 페이지로 이동 |

### 인터랙션

- 이메일/비밀번호 빈 상태로 로그인 클릭 시 입력 필드 하단에 빨간색 에러 메시지
- 로그인 실패 시 "이메일 또는 비밀번호가 올바르지 않습니다" 토스트
- 소셜 로그인 클릭 시 해당 OAuth2 제공자 인증 페이지로 리다이렉트
- 로그인 성공 시 메인 캘린더 (월간 뷰)로 이동

### 모바일 (768px 이하)

- 카드 너비 100%, 좌우 패딩 24px
- 소셜 로그인 버튼 세로 배치 유지
- 입력 필드 높이 48px (터치 영역 확보)

### 구성 요소

| 요소 | 설명 |
|------|------|
| 이메일 입력 | 실시간 중복 검사 (debounce 300ms), 형식 유효성 |
| 비밀번호 입력 | 8자 이상, 영문+숫자+특수문자, 강도 표시 바 |
| 비밀번호 확인 | 실시간 일치 여부 표시 (초록 체크 / 빨간 X) |
| 닉네임 입력 | 2~16자, 실시간 중복 검사 |
| 가입하기 버튼 | Warm Gold 배경, 모든 필드 유효 시에만 활성화 |
| 로그인 링크 | 로그인 페이지로 이동 |

### 인터랙션

- 각 필드 포커스 아웃 시 유효성 검증, 에러 시 필드 하단 빨간 텍스트
- 가입 성공 시 자동 로그인 후 메인 캘린더로 이동
- 개인 캘린더가 자동 생성됨

### 모바일

- 로그인 페이지와 동일한 레이아웃 전략

### 구성 요소

| 요소 | 설명 |
|------|------|
| **헤더** | |
| 로고 | 좌측 Memoria 로고, 클릭 시 오늘 날짜로 이동 |
| 뷰 전환 | 월간/주간 토글 버튼 (세그먼트 컨트롤 스타일) |
| 월 네비게이션 | ◀ ▶ 화살표로 이전/다음 월 이동, 중앙에 "YYYY년 M월" |
| 오늘 버튼 | 클릭 시 현재 날짜로 스크롤 |
| 알림 벨 | 미읽은 알림 개수 배지, 클릭 시 알림 드롭다운 |
| 프로필 | 아바타 이미지, 클릭 시 드롭다운 (프로필/설정/로그아웃) |
| **사이드바** | |
| 캘린더 리스트 | 체크박스로 캘린더 표시/숨김 토글, 각 캘린더 색상 표시 |
| 캘린더 설정 | ⚙ 아이콘 클릭 시 캘린더 설정 페이지로 이동 |
| 캘린더 만들기 | [+] 버튼 클릭 시 캘린더 생성 모달 |
| 오늘의 일정 | 오늘 날짜의 이벤트 요약 리스트 |
| AI 비서 버튼 | 클릭 시 AI 비서 사이드 패널 열기 (프리미엄) |
| **캘린더 그리드** | |
| 요일 헤더 | 일~토 (일요일 빨간색, 토요일 파란색) |
| 날짜 셀 | 클릭 시 해당 날짜에 이벤트 생성 모달 오픈 |
| 이벤트 바 | 캘린더 색상의 바 형태, 여러 날 이벤트는 셀을 가로질러 표시 |
| 이벤트 표시 | 셀 내 최대 3개, 초과 시 "+2 more" 클릭으로 확장 |
| 오늘 날짜 | Warm Gold 원형 배경으로 강조 |
| FAB | 우측 하단 고정, [+] 클릭 시 이벤트 생성 모달 |

### 인터랙션

- 날짜 셀 클릭 → 이벤트 생성 모달 (해당 날짜 자동 입력)
- 이벤트 바 클릭 → 이벤트 상세 모달
- 이벤트 드래그 앤 드롭 → 날짜 변경 (P1)
- 사이드바 캘린더 체크박스 토글 → 해당 캘린더 이벤트 즉시 표시/숨김
- 실시간 동기화: 다른 멤버가 이벤트 생성/수정 시 WebSocket으로 즉시 반영

### 모바일 (768px 이하)

- 사이드바 숨김 → 햄버거 메뉴(☰)로 슬라이드 인
- 캘린더 그리드 전체 너비
- 날짜 셀 높이 축소, 이벤트는 도트(●)로만 표시
- 날짜 탭 시 해당 날짜 이벤트 리스트가 하단에 펼쳐짐
- FAB 위치 우측 하단 유지

- 일정 카드의 "수정" 클릭 → 이벤트 생성 모달 오픈 (프리필된 상태)
- 충돌 감지 시 대안 시간 버튼 → 해당 시간으로 이벤트 수정 제안
- 일일 사용량 초과 시 입력 비활성 + "내일 다시 이용해 주세요" 메시지
- Free 사용자가 AI 버튼 클릭 시 → 프리미엄 업그레이드 유도 모달
