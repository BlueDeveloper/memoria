## 1. 다이어리 시스템

### 1.1 다이어리란?
- 사용자가 만드는 **독립적인 일정/메모 공간**
- 각 다이어리는 고유한 **로고, 테마, 색상**을 가짐
- 커플 다이어리, 가족 다이어리, 업무 다이어리, 웨딩 플래너 등 용도별 생성

### 1.2 다이어리 정책

| 항목 | 무료 | 유료 (티켓) |
|------|------|-------------|
| 다이어리 생성 | **1개** (가입 시 자동 생성) | 추가 1개당 N 티켓 |
| 기본 로고 | 제공 | - |
| 프리미엄 로고 | - | N 티켓 (영구 소장) |
| 기본 테마 | 3종 제공 | - |
| 프리미엄 테마 | - | N 티켓 (영구 소장) |
| 일반 다이어리 기능 | 전체 무료 | - |
| 웨딩 플래너 | 기본 기능 무료 | 고급 기능 N 티켓 |

### 1.3 다이어리 속성

| 필드 | 설명 |
|------|------|
| 이름 | 다이어리 제목 (예: "우리 커플 다이어리") |
| 로고 | 다이어리 표지 아이콘 (기본 / 프리미엄) |
| 테마 | 배경색, 폰트, UI 스타일 (기본 / 프리미엄) |
| 유형 | GENERAL, COUPLE, FAMILY, TEAM, WEDDING |
| 색상 | 대표 색상 |
| 멤버 | 공유 대상 (초대로 추가) |

## 2. 인트로 화면 (다이어리 선택)

### 사용자 플로우
```
앱/웹 접속
  → 다이어리 1개: 바로 해당 다이어리 진입 (인트로 스킵)
  → 다이어리 2개 이상: 인트로 화면에서 선택
```

### 초기 사용자 (다이어리 1개)
- 가입 시 기본 다이어리 1개 자동 생성 ("나의 다이어리")
- 접속하면 인트로 없이 바로 다이어리 뷰로 진입
- 사이드바에서 "+ 다이어리 추가" 가능 (티켓 필요)

### 인트로 화면 (다이어리 2개 이상일 때만)
- 상단: "Memoria" 로고 + 날짜/인사말
- 중앙: 다이어리 카드 그리드 (로고 + 이름 + 미리보기)
  - 각 카드에 오늘의 일정 수 배지 표시
  - "+" 버튼으로 새 다이어리 추가 (티켓 필요)
- 하단: 프로필 + 설정 + 티켓 잔액

### 비로그인 시
- 샘플 다이어리 1개로 바로 둘러보기 진입
- 일정 작성 시 → 회원가입 유도 모달

## 3. 티켓 시스템 (인앱 화폐)

### 3.1 티켓이란?
- Memoria 내 **모든 유료 아이템을 구매하는 통합 화폐**
- 실제 결제(카드/인앱결제)로 충전
- 충전한 티켓으로 아이템 구매 → **영구 소장**

### 3.2 티켓 충전

| 패키지 | 티켓 수 | 가격 (예정) | 보너스 |
|--------|---------|------------|--------|
| 소 | 10 티켓 | 1,000원 | - |
| 중 | 55 티켓 | 5,000원 | +5 보너스 |
| 대 | 120 티켓 | 10,000원 | +20 보너스 |
| 특대 | 300 티켓 | 20,000원 | +100 보너스 |

### 3.3 티켓 사용처

| 아이템 | 티켓 (예정) | 비고 |
|--------|------------|------|
| 추가 다이어리 (3개째~) | 30 | 영구 |
| 프리미엄 로고 | 5~15 | 영구 소장 |
| 프리미엄 테마 | 10~30 | 영구 소장 |
| 웨딩 플래너 고급 기능 | 50 | 영구 |
| AI 일정 비서 (월간) | 20/월 | 구독형 |

### 3.4 결제 연동
- **웹**: Toss Payments (카드 결제)
- **Android**: Google Play Billing
- **iOS**: Apple In-App Purchase

## 4. 다이어리 내부 기능 (기존 유지)

각 다이어리 내부는 기존 캘린더 기능 유지:
- 월간/주간 뷰
- 이벤트 CRUD (제목, 날짜/시간, 장소, 메모, 반복, 알림)
- 이벤트 댓글
- 멤버 초대 (카톡/이메일)
- 멤버 색상 구분

---

## 5. 웨딩 플래너 다이어리

### 무료 기능
- 기본 캘린더 (D-day 카운트다운)
- 체크리스트 (예식장, 스드메, 혼수 등)
- 예산 관리 (항목별 예산/실비 입력)

### 유료 기능 (티켓 구매)
- 웨딩 전문 템플릿 (타임라인, 좌석배치)
- AI 추천 (업체 추천, 일정 자동 배치)
- 프리미엄 초대장 디자인

## 6. 로고 & 테마 상점

### 로고 카테고리
- 기본 (무료): 5종 — 기본 다이어리, 하트, 별, 나무, 집
- 시즌 (유료): 벚꽃, 해변, 단풍, 눈꽃 등
- 캐릭터 (유료): 동물, 이모티콘 스타일
- 프리미엄 (유료): 아티스트 콜라보, 한정판

### 테마 카테고리
- 기본 (무료): 라이트, 다크, 세피아
- 파스텔 (유료): 핑크, 민트, 라벤더, 피치
- 시즌 (유료): 봄/여름/가을/겨울
- 프리미엄 (유료): 아티스트 콜라보

## 8. Phase 계획 (수정)

### Phase 1 — 웹 MVP
```
인증 → 인트로(다이어리 선택) → 다이어리 생성(2개 무료)
→ 다이어리 내 캘린더(월간/주간) → 이벤트 CRUD
→ 멤버 초대 → 기본 로고/테마
```

### Phase 1.5 — 수익화
```
티켓 충전 (Toss Payments) → 상점 (로고/테마)
→ 추가 다이어리 구매 → 웨딩 플래너 기본
```

### Phase 2 — 모바일 앱
```
React Native → 인앱 결제 (Play/Apple)
→ 푸시 알림 → AI 일정 비서
```
