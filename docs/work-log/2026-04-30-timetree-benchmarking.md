# TimeTree(타임트리) 벤치마킹 분석

> 작성일: 2026-04-30
> 목적: Memoria 프로젝트 기획을 위한 타임트리 서비스 분석

---

## 1. 핵심 기능 목록

### 1.1 캘린더 관리
- **무제한 공유 캘린더**: 가족, 커플, 친구, 팀 등 그룹별로 별도 캘린더 생성 가능
- **개인 캘린더**: 공유하지 않는 개인 일정용 캘린더 별도 운영
- **Public Calendar**: 관리자가 생성하여 불특정 다수에게 공개하는 캘린더
- **캘린더 필터링**: 여러 캘린더를 통합 보기하거나 선택적으로 전환 가능

### 1.2 이벤트 관리
- **이벤트 생성/수정/삭제**: 제목, 날짜/시간, 장소, 메모 입력
- **색상 라벨링**: 멤버별 또는 활동별 색상 코딩 (라벨 이름 커스터마이징 가능)
- **반복 이벤트**: 매일/매주/매월/매년 반복 설정 (단, 매월 둘째 화요일 같은 복잡한 반복은 미지원)
- **D-Day 기능**: 특정 이벤트까지 남은 일수 표시
- **음력 지원**: 음력 날짜 기반 이벤트 설정
- **이벤트 고정(Pin)**: 중요 이벤트를 캘린더 상단에 고정 (프리미엄)
- **파일 첨부**: PDF, Word, 사진 등을 이벤트에 첨부 (프리미엄)
- **사진 공유**: 이벤트에 사진 첨부하여 시각적 맥락 제공

### 1.3 커뮤니케이션
- **이벤트 댓글**: 각 이벤트 하단에서 참여자 간 대화 가능
- **그룹 채팅**: 캘린더 단위로 그룹 채팅 지원 (별도 메신저 앱 불필요)
- **활동 피드**: 캘린더 내 변경사항을 피드 형태로 확인

### 1.4 메모 & 할 일
- **Keep 기능 (공유 메모)**: 캘린더에 메모를 작성하여 멤버와 공유
- **To-Do 리스트**: 할 일 목록 작성 및 공유
- **쇼핑 리스트**: 장보기 목록 등 리스트형 메모

### 1.5 알림
- **이벤트 알림 설정**: 이벤트 시작 전 알림 (시간 단위 설정)
- **푸시 알림**: 멤버의 일정 추가/수정 시 알림

### 1.6 외부 연동
- **Google Calendar 동기화**
- **Apple Calendar 동기화**
- **Outlook Calendar 동기화**
- **외부 캘린더 색상 커스터마이징**: 연동된 외부 캘린더의 표시 색상 변경 가능

### 1.7 그룹 관리
- **초대 시스템**: 이메일, Facebook Messenger, WhatsApp, SMS, 초대 링크, QR코드
- **멤버 프로필**: 각 멤버별 프로필 및 색상 지정
- **다중 초대**: 초대 링크로 여러 명 동시 초대 가능

### 1.8 기타
- **위젯 지원**: 홈 화면 위젯
- **크로스 플랫폼**: iOS, Android, Web 지원 (데스크톱 전용 앱은 없음)

## 2. UI/UX 특징

### 2.1 뷰 타입
| 뷰 | 설명 |
|------|------|
| **월간 뷰 (Monthly)** | 기본 뷰. 월 단위로 일정을 한눈에 확인. 하단에 메모 표시 |
| **주간 뷰 (Weekly)** | 주 단위 일정 확인 |
| **목록 뷰 (List)** | 일정을 리스트 형태로 나열 |
| **세로 뷰 (Vertical)** | 최대 3일간의 일정을 시간대별로 세로 표시 (프리미엄 전용) |

### 2.2 네비게이션 구조
- **상단 좌측**: 캘린더 이미지 아이콘 (캘린더 전환)
- **하단 우측**: 설정(Settings) 메뉴
- **탭 기반 네비게이션**: 캘린더, 메모, 채팅 등 주요 기능 간 이동
- **필터 기능**: 특정 캘린더/라벨만 선택하여 표시

### 2.3 색상 체계
- **멤버별 색상**: 각 멤버에게 고유 색상을 지정하여 누구의 일정인지 한눈에 식별
- **라벨 색상**: 활동 유형별 색상 지정 가능 (예: 업무=파랑, 개인=초록)
- **라벨 관리**: 설정 > Label Management에서 색상/이름/순서 커스터마이징
- **마지막 선택 색상 기억**: 이벤트 생성 시 직전 사용 색상 자동 선택
- **전체적 디자인 톤**: 깔끔하고 부드러운 인터페이스, 기업용이 아닌 개인/가족 친화적 느낌

### 2.4 디자인 특성
- **직관적 인터페이스**: 모든 연령대가 쉽게 사용 가능한 단순한 UI
- **깔끔한 레이아웃**: 필수 정보만 한눈에 표시, 시각적 복잡성 최소화
- **비기업적 디자인**: 따뜻하고 친근한 느낌 (Google Calendar의 생산성 도구 느낌과 차별화)

## 3. 사용자 플로우

### 3.1 회원가입
```
앱 다운로드 → 시작 화면
  ├─ 이메일로 가입
  ├─ Facebook으로 가입
  └─ "건너뛰기(Skip to start)" → 나중에 계정 생성 가능
```

### 3.2 캘린더 생성
```
가입 완료 → 캘린더 생성 안내
  ├─ 공유 캘린더 생성 (가족, 커플, 팀 등 그룹 유형 선택)
  ├─ 개인 캘린더 생성
  └─ Public 캘린더 생성 (관리자용)
```

### 3.3 멤버 초대
```
캘린더 생성 완료 →
  1. 좌측 상단 캘린더 아이콘 탭
  2. 초대할 캘린더 선택
  3. 우측 하단 "설정" 탭
  4. "멤버 초대" 탭
  5. 초대 방법 선택:
     ├─ 이메일
     ├─ Facebook Messenger
     ├─ WhatsApp
     ├─ SMS
     ├─ 초대 링크 (다수 초대용)
     └─ QR 코드
```

### 3.4 초대 수락 (피초대자)
```
초대 URL 수신 또는 QR 코드 스캔
  → 앱 다운로드 (미설치 시)
  → "이 캘린더에 참여" 탭
  → 프로필 선택
  → 공유 캘린더 합류 완료
```

### 3.5 일정 관리
```
캘린더 화면 → 날짜 선택 → "+" 버튼
  → 이벤트 생성:
     ├─ 제목 입력
     ├─ 날짜/시간 설정
     ├─ 장소 입력
     ├─ 라벨(색상) 선택
     ├─ 알림 설정
     ├─ 반복 설정
     ├─ 메모 입력
     └─ 파일/사진 첨부 (프리미엄)
  → 저장 → 멤버에게 자동 알림
```

## 4. 수익 모델

### 4.1 프리미엄 구독 (TimeTree Premium)
| 항목 | 내용 |
|------|------|
| **월간 구독** | $4.49 USD/월 (첫 1개월 무료) |
| **연간 구독** | $44.99 USD/년 (첫 1개월 무료, 2개월 할인) |
| **과금 단위** | 사용자당 (per user) |
| **출시일** | 2022년 4월 |

### 4.2 프리미엄 전용 기능
- 광고 제거
- 파일 첨부 (PDF, Word, 사진 등)
- 이벤트 고정 (Pin)
- 세로 뷰 (Vertical View)
- 우선 고객 지원
- 이벤트 정렬

### 4.3 광고 수익
- **TimeTree Ads (MIRAI_DSP)**: 무료 사용자에게 앱 내 광고 표시
- 일정 작성 화면 등에 배너 광고 노출

### 4.4 무료 vs 프리미엄 비교
| 기능 | 무료 | 프리미엄 |
|------|------|---------|
| 공유 캘린더 | 무제한 | 무제한 |

| 채팅/댓글 | O | O |
| 메모/To-Do | O | O |
| 외부 캘린더 연동 | O | O |
| 광고 | 있음 | 없음 |
| 파일 첨부 | X | O |
| 이벤트 고정 | X | O |
| 세로 뷰 | X | O |

## 5. 강점과 약점

### 5.1 강점

| 항목 | 설명 |
|------|------|
| **공유 특화 설계** | 개인 일정 관리가 아닌 "함께 쓰는 캘린더"라는 명확한 포지셔닝. 일정 공유의 패러다임을 바꿈 |
| **직관적 UI** | 모든 연령대가 쉽게 사용 가능한 단순하고 깔끔한 인터페이스 |
| **통합 커뮤니케이션** | 이벤트 댓글, 그룹 채팅 내장으로 별도 메신저 불필요 |
| **관대한 무료 버전** | 핵심 기능 대부분 무료. 무제한 공유 캘린더 생성 |
| **크로스 플랫폼** | iOS, Android, Web 모두 지원 |
| **외부 캘린더 연동** | Google, Apple, Outlook 캘린더 동기화 |
| **색상 코딩 시스템** | 멤버별/활동별 색상 지정으로 시각적 구분 용이 |
| **글로벌 사용자 기반** | 2024년 10월 기준 6,000만 명 이상 등록 사용자 |
| **다양한 초대 방법** | 링크, QR코드, 이메일, 메신저 등 다채널 초대 |

### 5.2 약점

| 항목 | 설명 |
|------|------|
| **반복 일정 제한** | 매월 둘째 화요일, 특정 날짜까지 반복 등 복잡한 반복 패턴 미지원 |
| **동기화 지연** | 기기 간 동기화가 즉시 이루어지지 않고 수분 소요되는 경우 보고 |
| **광고 크기** | 무료 버전의 광고가 크고 미관을 해친다는 사용자 불만 |
| **프라이버시 제한** | 개인정보보호 설정이 제한적, E2E 암호화 미지원 |
| **데스크톱 앱 부재** | 웹 버전만 존재, 전용 데스크톱 앱 없음 |
| **작업 관리 미흡** | 본격적인 태스크 관리 기능(칸반, 우선순위 등) 부족 |
| **통합 기능 제한** | 이메일, 생산성 도구(Slack, Notion 등)와의 연동 약함 |
| **스마트 스케줄링 부재** | AI 기반 자동 일정 제안, 빈 시간 탐색 등 고급 자동화 미지원 |
| **화상 회의 미지원** | Zoom, Google Meet 등과의 통합 없음 |
| **프리미엄 가격 논란** | 연 $44.99(한화 약 3만원)의 가치에 의문을 제기하는 사용자 존재 |
| **리뉴얼 후 디자인 아쉬움** | 색상 테마 축소, 반투명 UI 변경 등에 대한 기존 사용자 불만 |

## 6. Memoria 프로젝트를 위한 시사점

### 벤치마킹 포인트 (따라할 것)
1. **공유 중심 설계**: 캘린더 공유를 핵심 가치로 명확히 포지셔닝
2. **통합 커뮤니케이션**: 이벤트 내 댓글/채팅으로 소통 허들 제거
3. **관대한 무료 정책**: 핵심 기능은 무료로 제공하여 사용자 확보
4. **다양한 초대 방법**: 링크, QR코드 등 접근 장벽 최소화
5. **색상 코딩 시스템**: 멤버별 색상으로 직관적 일정 구분

### 차별화 포인트 (개선할 것)
1. **고급 반복 일정**: 복잡한 반복 패턴(매월 N째 X요일, 특정 날짜까지 등) 지원
2. **실시간 동기화**: WebSocket 등을 활용한 즉시 동기화
3. **강화된 프라이버시**: E2E 암호화, 세밀한 권한 관리
4. **데스크톱 앱**: PWA 또는 Electron 기반 데스크톱 지원
5. **스마트 스케줄링**: AI 기반 빈 시간 탐색, 일정 제안
6. **외부 도구 통합**: Slack, Notion, Zoom 등과의 연동
7. **광고 UX 개선**: 광고 삽입 시 사용자 경험 최소 침해 설계
